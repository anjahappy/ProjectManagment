# **生活场景故事**
| **中大人IT生活攻略**   | **中大人的移动生活攻略**   | 
|:----|:----|
| **入校前:**   | 入校前   | 
| 在我们入校之前，我们的新生主要依靠了解学校官网和自己的学院官网了解信息。   | 得到通知书后，通过移动门户，一步了解校园周边服务，校园概况以及校园社团的信息，方便快捷找到自己想要的服务。   | 
| **关于Netid**   | 1-3年级   | 
| 在大一入校过程中，我们每个人都会分配一个专属的Netid，这个账户可以连接我们在学校的很多业务，包括大学服务中心服务、校园卡信息服务、图书馆信息服务、通过购买校园网以后，可以访问教务系统等。另外也会分配一个校园邮箱，这个校园邮箱帮助我们可以更好的从事学习交流，导师联系等一系列工作。这个邮箱也和微信企业邮箱绑定，我们使我们可以利用腾讯相关资源。   | 进入校园后，通过移动门户可以完成大部分自己需要的服务。需要查找各种资料时，通过移动门户一键搜索，展示图书馆相关书籍，并提供电子书服务。新学期选课时，通过移动门户一键查看自己喜欢的课程，并利用app自动抢课，并直接展示自己的课程表。  每一门课程的对应资源及网站也可以直接点击课表对应课程进入，上课签名也一样的方便快捷。学校网站一键下载各种正版已购app，并且提供校内网络加速。  同学之间互相共享资源，书籍、电影、学习资料、英语资料、课程等，都可以通过移动门户共享云盘进行分享，共享云盘，通过p2p保存在全校每一台设备上，无下载流量限制，无速度限制。  生活上无论是校园卡充值，还是校车查看，现在都可以在移动门户内的生活模块下查看，一键充值，预定，与学校官方的模块连接。  移动门户还会自动监视你的网络费用，提醒充值，需要订水时也只需要在门户下订即可，宿舍里有需要维修时，也只需要点击预约维修。  移动门户会罗列、推荐校园里正在进行的各项活动，马拉松，社团游戏，实验助理，各类宣讲会，名师演讲等，一键即可预约参与。门户还提供了树洞服务，同学们可以寻找自己想要交流的校友，匿名或者不匿名的大胆交谈，说出自己的心意。       | 
| **大一-大四期间服务**   | 4年级   | 
| 在大一到大四的学习过程中我们已经可以利用中山大学于我们提供的各类软件进行科研学习，这包括微软服务、教学题库系统（MOJ\SOJ\go-online\VMatrix）、office、MATLAB、福昕等服务。微软服务、福昕我们可以免费使用office，利用编辑阅读器来观看制作PPT。  学校的每个学院以及各组织都有各自的官网，也创建了各自的官方微信公众号，我们利用自己的Netid账户以及所绑定的校园卡可以申请学校的各个设施资源，包括体育资源、教室资源、场地资源。可以使用学校学院组织的各类微信公众号来查询办理相关业务  由于疫情原因，中山大学已经和知网等学术资源网站进行合作，为避免个别地方VPN不稳定，提供了选择学校跳转Netid登录方式。使校外学生不再为VPN网络问题担忧。另外可以使用Skype for Business进行线上会议，zoom.edu.cn 视频会议服务可以实现多人高清音视频会议、远程直播教学实时交互、数据信息同步共享，具备共享 Office 和 PDF 文档、应用程序、音视频文件、桌面、会议录制、文字聊天、快速邀请等功能。目前我们的授课方式已经收疫情影响，由线下转移到线上。我们学校有专门的客户端，通过客户端我们每天都可以实时汇报健康情况，使用查询成绩等相关服务。  其中关于校外使用服务需要连接校园网VPN，校内连上校园网即可使用。其中校园网资费标准是30元\月和300元\年两种。  关于校园网连接问题，有多种解决方法，在校内无线网我们可以直接使用Netid登录，校内有线网可以按照服务帮助台指引，安装相应客户端进行网络练级。在校外支持easyconnect客户端连接VPN服务。   | 移动门户还为四年级同学准备了特色服务  当我们进入四年级后，门户会制定三个内容：考研、留学、工作。  每一个模块下面都有丰富详细的内容，总结了所有师兄师姐的的方法技巧。  我们无论是想要提问还是，搜寻往年的资料，移动门户都进行了细致的分类和整理。  同时，所有想要进行毕业旅行的同学也都可以通过门户来组队，一键组队，选择和自己目的地相同的同学一起去旅行吧。   | 
| **毕业后**   | 毕业后   | 
| 毕业后我们可以有校友会以及相应的校友联络办公室，对于我们SDCS数据科学与计算机学院有职业发展与校友工作办公室进行联系。其也有相应的微信公众号可以提供相关网络信息服务   | 毕业后的我们，移动门户会自行更新为校友模式，无论是校友联系、帮助学校或是校友交流，都可以同国这个门户进行，实时查看学校动态，了解校园里的信息。   |
